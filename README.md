# 메모 타임스탬프 앱

개인 메모 작성 및 관리 웹 애플리케이션입니다. Firebase를 통한 실시간 동기화를 지원하여 여러 기기에서 동일한 데이터를 사용할 수 있습니다.

## 🚀 주요 기능

- ✍️ **메모 작성**: 텍스트와 이미지를 포함한 메모 작성
- 💬 **타래(답글) 기능**: 메모에 답글을 달아 대화형 구조 생성
- 📁 **폴더 관리**: 메모를 폴더로 분류하여 관리
- 🔍 **검색**: 메모 내용 검색
- 📅 **달력 필터**: 날짜별로 메모 필터링
- 👤 **프로필 관리**: 이름, 소개, 프로필 사진 설정
- ☁️ **Firebase 동기화**: 여러 기기에서 실시간 동기화
- 🔄 **계정 전환**: 여러 사용자 계정 간 전환 가능
- 📱 **모바일 지원**: 모바일 기기 최적화 인터페이스

## 🎯 시작하기

### 1. 사용자 ID 설정

처음 웹사이트를 열면 사용자 ID 입력 모달이 표시됩니다.

1. 사용자 ID를 입력하세요 (예: `user1`, `myaccount`)
2. "확인" 버튼을 클릭하거나 Enter 키를 누르세요
3. 입력한 ID는 자동으로 저장되며, 이후 방문 시 자동으로 사용됩니다

**💡 팁**: 같은 ID를 사용하는 모든 기기에서 데이터가 자동으로 동기화됩니다.

### 2. 프로필 설정

왼쪽 프로필 섹션에서:
- **이름**: 프로필 이름 입력란에 이름을 입력하세요
- **소개**: 소개 입력란에 간단한 소개를 작성하세요
- **프로필 사진**: 프로필 이미지 영역을 우클릭하여 사진을 추가하거나 삭제할 수 있습니다

## 📝 기본 사용법

### 메모 작성하기

1. 중앙의 글 작성 영역(composer)에 텍스트를 입력하세요
2. "전송" 버튼(▶)을 클릭하거나 `Ctrl + Enter`를 누르면 메모가 저장됩니다
3. 메모는 작성 시간과 함께 목록에 표시됩니다

### 이미지 첨부하기

1. "사진 첨부" 버튼(📷)을 클릭하세요
2. 이미지 파일을 선택하세요 (여러 개 선택 가능)
3. 선택한 이미지는 미리보기로 표시됩니다
4. 이미지 미리보기에서 "X" 버튼을 클릭하면 제거됩니다
5. 메모를 전송하면 이미지가 함께 저장됩니다

**💡 팁**: 이미지는 드래그 앤 드롭으로 순서를 변경할 수 있습니다.

### 메모 보기

- 저장된 메모는 글 작성 영역 아래에 시간순으로 표시됩니다
- 이미지가 포함된 메모는 썸네일로 표시되며, 클릭하면 전체 크기로 볼 수 있습니다
- 여러 이미지가 있는 경우 화살표(‹ ›)로 이전/다음 이미지를 볼 수 있습니다

## 🎨 고급 기능

### 타래(답글) 기능

메모에 답글을 달아 대화형 구조를 만들 수 있습니다.

1. 메모를 **우클릭**하세요
2. "답글" 메뉴를 선택하세요
3. 글 작성 영역 상단에 "답글 작성 중" 표시가 나타납니다
4. 답글 내용을 작성하고 전송하세요
5. 답글은 원본 메모 아래에 들여쓰기되어 표시됩니다

**💡 팁**: 답글에도 다시 답글을 달 수 있어 중첩 구조를 만들 수 있습니다.

### 폴더 관리

메모를 폴더로 분류하여 관리할 수 있습니다.

#### 폴더 추가하기

1. 오른쪽 사이드바에서 "폴더" 버튼을 클릭하세요
2. "추가" 버튼을 클릭하세요
3. 폴더 이름을 입력하고 Enter 키를 누르세요

#### 메모를 폴더에 추가하기

**방법 1: 우클릭 메뉴 사용**
1. 메모를 우클릭하세요
2. "폴더 추가" 메뉴를 선택하세요
3. 추가할 폴더를 선택하세요

**방법 2: 선택 모드 사용**
1. "선택" 버튼을 클릭하여 선택 모드를 활성화하세요
2. 메모 옆의 체크박스를 클릭하여 메모를 선택하세요
3. "폴더" 버튼을 클릭하세요
4. 추가할 폴더를 클릭하세요

#### 폴더에서 메모 보기

1. "폴더" 버튼을 클릭하여 폴더 목록을 엽니다
2. 원하는 폴더를 클릭하면 해당 폴더의 메모만 표시됩니다
3. "필터 해제" 버튼을 클릭하면 전체 메모를 다시 볼 수 있습니다

#### 폴더에서 메모 해제하기

1. "선택" 버튼을 클릭하여 선택 모드를 활성화하세요
2. 해제할 메모들을 선택하세요
3. "폴더" 버튼을 클릭하세요
4. "해제" 버튼을 클릭하세요
5. 선택한 메모들이 모든 폴더에서 해제됩니다

#### 폴더 수정/삭제하기

1. "폴더" 버튼을 클릭하세요
2. 수정/삭제할 폴더를 클릭하여 선택하세요
3. "수정" 버튼을 클릭하여 이름을 변경하거나
4. "삭제" 버튼을 클릭하여 폴더를 삭제하세요

**💡 참고**: 
- "북마크" 폴더와 "사진" 폴더는 자동으로 생성되며 삭제할 수 없습니다
- 메모의 시간을 클릭하면 북마크 폴더에 자동으로 추가/제거됩니다
- 사진이 포함된 메모는 자동으로 사진 폴더에 표시됩니다

### 검색 기능

1. 오른쪽 사이드바의 검색창에 검색어를 입력하세요
2. Enter 키를 누르거나 검색 버튼을 클릭하세요
3. 검색어가 포함된 메모만 표시됩니다
4. 검색창 옆의 X 버튼을 클릭하면 검색이 해제됩니다

**💡 팁**: 검색 결과에는 검색어가 포함된 메모와 그 메모의 타래(답글)도 함께 표시됩니다.

### 달력 필터

날짜별로 메모를 필터링할 수 있습니다.

1. 달력에서 원하는 날짜를 클릭하세요
2. 해당 날짜에 작성된 메모만 표시됩니다
3. Shift 키를 누른 채로 날짜를 클릭하면 날짜 범위를 선택할 수 있습니다
4. "필터 해제" 버튼을 클릭하면 필터가 해제됩니다

### 선택 모드 및 일괄 작업

여러 메모를 한 번에 선택하여 작업할 수 있습니다.

1. "선택" 버튼을 클릭하여 선택 모드를 활성화하세요
2. 메모 옆의 체크박스를 클릭하여 메모를 선택하세요
3. "전체" 버튼을 클릭하면 모든 메모가 선택됩니다
4. 선택한 메모에 대해 다음 작업을 수행할 수 있습니다:
   - **수정**: "수정" 버튼을 클릭하여 여러 메모를 동시에 편집
   - **삭제**: "삭제" 버튼을 클릭하여 선택한 메모를 삭제
   - **폴더 추가**: 폴더에 추가
   - **폴더 해제**: 모든 폴더에서 해제

### 투명화 기능

메모 텍스트를 투명하게 만들어 프라이버시를 보호할 수 있습니다.

1. "투명" 버튼을 클릭하세요
2. 모든 메모의 텍스트가 투명해집니다
3. 다시 클릭하면 원래대로 돌아갑니다

**💡 팁**: 모바일에서는 글 작성 영역 왼쪽에 투명 버튼이 표시됩니다.

### 날짜 표시 모드

메모의 시간 표시 형식을 변경할 수 있습니다.

1. "날짜" 버튼을 클릭하세요
2. 시간이 `mm/dd hh:mm` 형식으로 표시됩니다 (예: `12/25 14:30`)
3. 다시 클릭하면 `hh:mm` 형식으로 돌아갑니다 (예: `14:30`)

### 초기화 및 복원

모든 데이터를 초기화하거나 복원할 수 있습니다.

#### 초기화하기

1. "초기화" 버튼을 클릭하세요
2. 확인 모달에서 "네"를 클릭하세요
3. 모든 메모, 프로필, 폴더가 삭제됩니다
4. Firebase의 데이터도 함께 삭제됩니다

**⚠️ 주의**: 초기화는 되돌릴 수 없습니다. 신중하게 사용하세요.

#### 복원하기

초기화 후에는 다음 옵션이 표시됩니다:
- **복원**: 초기화 전 상태로 되돌립니다
- **진짜끝**: 복원을 포기하고 데이터를 영구적으로 삭제합니다

## 📱 모바일 사용법

모바일 기기에서는 간소화된 인터페이스가 제공됩니다.

### 모바일 화면 구성

- **글 작성 영역**: 상단에 표시됩니다
- **메모 목록**: 글 작성 영역 아래에 표시됩니다
- 프로필, 달력, 검색 등은 모바일에서는 숨겨집니다

### 모바일 전용 기능

#### 메모 컨텍스트 메뉴

메모를 **길게 누르면** 다음 메뉴가 표시됩니다:
- **수정**: 메모 내용 수정
- **삭제**: 메모 삭제
- **답글**: 메모에 답글 달기
- **폴더 추가**: 메모를 폴더에 추가

#### 투명화 버튼

글 작성 영역 왼쪽에 투명 버튼이 표시되어 프라이버시를 보호할 수 있습니다.

## 👤 사용자 ID 관리

### 사용자 ID 변경하기

1. 화면 상단의 하얀색 헤더 오른쪽 위에 있는 "다른 ID" 버튼을 클릭하세요
2. 변경할 사용자 ID를 입력하세요
3. "확인" 버튼을 클릭하거나 Enter 키를 누르세요

### 계정 전환 vs 데이터 마이그레이션

**계정 전환** (기존 ID가 Firebase에 존재하는 경우):
- 확인 모달이 표시됩니다
- "전환" 버튼을 클릭하면 해당 계정의 데이터를 사용합니다
- 기존 계정의 데이터는 그대로 유지됩니다

**데이터 마이그레이션** (새로운 ID인 경우):
- 현재 계정의 데이터가 새 ID로 복사됩니다
- 기존 ID의 데이터는 삭제됩니다

### 현재 사용자 ID 확인

프로필 섹션의 이름 아래에 현재 사용자 ID가 회색으로 표시됩니다.

## 🔄 Firebase 동기화

### 자동 동기화

- 메모를 작성하거나 수정하면 자동으로 Firebase에 저장됩니다
- 다른 기기에서 변경사항이 있으면 실시간으로 동기화됩니다
- 프로필 정보(이름, 소개, 사진)도 자동으로 동기화됩니다

### 동기화 상태 확인

브라우저 개발자 도구(F12)의 콘솔에서 동기화 상태를 확인할 수 있습니다:
- `Firebase 초기화 완료`: Firebase가 정상적으로 연결됨
- `Firebase에 저장 성공`: 데이터 저장 성공
- `실시간 동기화 적용`: 다른 기기에서 변경사항 수신

## 💡 유용한 팁

1. **키보드 단축키**:
   - `Ctrl + Enter`: 메모 전송
   - `Enter`: 줄바꿈 (답글 모드가 아닐 때)

2. **이미지 관리**:
   - 이미지 미리보기를 드래그하여 순서 변경 가능
   - 이미지 클릭 시 전체 크기로 보기
   - 여러 이미지가 있는 경우 화살표로 탐색

3. **타래 구조**:
   - 답글은 들여쓰기되어 표시됩니다
   - 타래가 있는 메모는 가장 최신 답글 시간 기준으로 정렬됩니다

4. **폴더 활용**:
   - 메모는 여러 폴더에 동시에 속할 수 있습니다
   - 북마크 폴더는 시간 클릭으로 빠르게 추가/제거 가능

5. **검색 팁**:
   - 검색어가 포함된 메모의 타래도 함께 표시됩니다
   - 검색창을 비우면 자동으로 검색이 해제됩니다

## ❓ FAQ

### Q: 메모가 저장되지 않아요
A: 브라우저 개발자 도구(F12)의 콘솔을 확인하세요. Firebase 연결 상태를 확인할 수 있습니다.

### Q: 다른 기기에서 메모가 보이지 않아요
A: 
1. 같은 사용자 ID를 사용하고 있는지 확인하세요
2. Firebase 연결 상태를 확인하세요
3. 브라우저를 새로고침해보세요

### Q: 사용자 ID를 잊어버렸어요
A: 프로필 섹션의 이름 아래에 현재 사용자 ID가 표시됩니다.

### Q: 초기화 후 복원할 수 있나요?
A: 네, 초기화 직후에는 "복원" 버튼으로 되돌릴 수 있습니다. "진짜끝" 버튼을 누르면 영구적으로 삭제됩니다.

### Q: 모바일에서 기능이 작동하지 않아요
A: 
1. 브라우저를 최신 버전으로 업데이트하세요
2. 페이지를 새로고침하세요
3. 개발자 도구 콘솔에서 오류 메시지를 확인하세요

### Q: 이미지가 표시되지 않아요
A: 
1. 이미지 파일 형식이 지원되는지 확인하세요 (JPG, PNG, GIF 등)
2. 파일 크기가 너무 크지 않은지 확인하세요
3. 브라우저 콘솔에서 오류 메시지를 확인하세요

### Q: Firebase 동기화가 안 돼요
A:
1. 인터넷 연결을 확인하세요
2. Firebase 콘솔에서 보안 규칙이 올바르게 설정되어 있는지 확인하세요
3. 브라우저 콘솔에서 오류 메시지를 확인하세요
