# Supabase로 전환 가이드

## Supabase 장점
- ✅ 무료 플랜: 500MB DB, 2GB 파일 스토리지, **무제한 API 요청**
- ✅ Firebase와 거의 동일한 API 구조
- ✅ 실시간 동기화 지원
- ✅ 오픈소스 기반

## 설정 방법

1. **Supabase 계정 생성**
   - https://supabase.com 접속
   - 무료 계정 생성

2. **프로젝트 생성**
   - 새 프로젝트 생성
   - 프로젝트 URL과 API Key 복사

3. **코드 변경**
   - `index.html`에서 Firebase SDK를 Supabase SDK로 변경
   - API 키만 교체하면 됨

## Supabase 무료 플랜 제한
- 데이터베이스: 500MB
- 파일 스토리지: 2GB
- API 요청: **무제한** ✅
- 실시간 연결: 200개 동시 연결

Firebase의 Quota exceeded 문제가 해결됩니다!
